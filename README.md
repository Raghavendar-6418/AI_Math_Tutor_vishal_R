# AI Math Tutor - Naan Mudhalvan Project

This project is an **AI-powered Math Tutoring Application** built for the **Naan Mudhalvan** skill development initiative. It helps students solve mathematical problems and understand concepts using artificial intelligence.

## 🚀 Features
* Step-by-step math problem solving.
* Vision-based problem detection.
* Interactive Student Learning UI.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit
* **AI Model:** Google Gemini AI
